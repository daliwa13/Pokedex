# Pokedex
Pokedex is a simple JS app that fetches data from public API pokeapi.co and displays them on a webpage.

## Instalation
Download the files to a folder and open index.html in a web browser of your choice.

## Tech Stack
- HTML 5
- CSS 3
- JavaScript
- pokeapi.co
